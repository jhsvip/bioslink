# 主板 BIOS 链接管理器

一个简洁美观、功能实用的桌面端工具，用于统一管理、快速查询、高效维护各大品牌主板的 BIOS 下载链接，适合电脑维修人员、装机技术员、硬件爱好者使用。

在线查询链接：https://jhsvip.github.io/
---

## ✨ 功能特性

- 🖥 **清爽可视化界面**：基于 Python Tkinter 开发，布局清晰、操作简单
- 🔍 **多条件组合查询**：支持按品牌、芯片组、系列、型号模糊搜索
- 📌 **重点信息突出展示**
  - 型号在最左侧、加宽、加粗显示
  - 品牌、芯片组自动缩进，层次更清晰
- ✏️ **完整数据管理**：添加、编辑、删除、查看、刷新
- 🔗 **双击打开链接**：直接调用浏览器打开 BIOS 下载地址
- 💾 **本地持久化存储**：自动保存 JSON 文件，数据不丢失
- 📤 **数据导出**：支持导出 TXT / JSON 格式
- 📏 **界面优化**：查询结果自动填满底部空间，体验舒适

---

## 🚀 快速开始

### 环境要求
- Python 3.x
- 无需安装第三方依赖

### 运行命令
python bios_manager.py

---

## 📌 界面布局说明

- **上半部分**：添加 BIOS 信息 + 查询筛选面板
- **下半部分**：查询结果表格（自动占满窗口）
- **型号列**：最左、最大、最醒目
- **品牌 / 芯片组**：自动缩进，层次分明
- **双击记录**：直接打开 BIOS 下载链接

## 📝 使用说明

1. 添加时 **品牌、型号为必填项**
2. 下载链接必须以 `http://` 或 `https://` 开头
3. 查询时品牌选择 **全部** 可显示所有数据
4. 双击表格可直接打开 BIOS 下载页面
5. 数据自动保存在 `bios_links.json`
6. 支持导出备份、编辑、删除
